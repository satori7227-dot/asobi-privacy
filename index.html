プライバシーポリシー

あそぶ？（Asobi）アプリ ・ 最終更新: 2026年6月18日
3行サマリー： あそぶ？は あなたから個人情報を一切集めません。 👍👎・お気に入りは 端末内のみ に保存され、運営者を含む誰にも送信されません。 「アイデア箱」だけが任意のメール送信で、これは あなた自身が送信ボタンを押した時のみ 動作します。

1. 運営者

本アプリは個人開発者「井戸川 聡里」が運営しています。 ご質問・ご要望は satori7227@gmail.com までお願いします。

2. 収集する情報

本アプリは以下の情報を 一切収集しません：

  • 氏名・メールアドレス・電話番号などの個人情報
  • 位置情報・連絡先・写真・カメラへのアクセス
  • 広告 ID・トラッキング ID
  • アプリ使用状況の解析・統計データ
  • クラッシュレポート・テレメトリ

3. 端末内に保存される情報

以下の情報は あなたの iPhone 端末内のみに保存 され、外部送信されません：

  • ゲームの「やる！／ちがう」フィードバック
  • お気に入りに登録したゲームの一覧
  • ツール（ルーレット等）に入力した名前リスト（セッション内のみ）

これらは iOS 標準の UserDefaults に保存され、アプリ削除と同時に消去されます。 iCloud バックアップを有効にしている場合、Apple の標準バックアップにのみ含まれます。

4. 「アイデア箱」機能について

「こんなゲームが欲しい」を投稿する機能は、あなたが 「送信」ボタンを押した時のみ 標準のメールアプリを起動し、宛先 satori7227@gmail.com に あなたが入力した文章を送信します（送信元はあなたのメールアプリ）。

受け取った内容は、運営者がゲームデータベースの改善・追加に使用します。 個人を特定する目的では使用しません。第三者に共有することはありません。

メールアプリが見つからない場合、内容をクリップボードにコピーします。 クリップボード上のデータは iOS 標準仕様により、他のアプリで貼り付け可能になります。 ご注意ください。

5. 第三者サービス・SDK

本アプリは以下を 使用していません：

  • Google Analytics・Firebase・Mixpanel 等の解析サービス
  • Facebook SDK・X (Twitter) SDK 等の SNS 連携
  • 広告ネットワーク・トラッキング SDK
  • 外部 API（クラウドサービスへのデータ送信）

6. お子様の利用について

本アプリは年齢制限なしで利用可能ですが、個人情報を収集しない設計のため、 13歳未満のお子様も保護者の同意のもとで安心して利用できます。

7. ポリシーの変更

本ポリシーを変更する場合は、本ページに変更内容を掲載します。 重大な変更がある場合はアプリ内でも通知します。
